Pseudo-classes em CSS

Introdução: o que são, para que servem e exemplos práticos.

O que são pseudo-classes?

Pseudo-classes são palavras-chave anexadas a seletores que aplicam estilos com base em estados ou condições, sem alterar o HTML.

Para que servem?

  • :hover, :focus (interação)
  • :first-child, :nth-child() (estrutura)
  • :not(), :disabled (condições)

Quando usar?

  • Feedback visual (hover, focus, active)
  • Estilos condicionais por posição
  • Regras avançadas sem alterar HTML

Exemplo: Hover em links

a {
  color: #0366d6;
  text-decoration: none;
}
a:hover {
  text-decoration: underline;
}

Exemplo: Foco em campo

input {
  border: 1px solid #ccc;
  padding: 0.5em;
  transition: border-color 0.2s;
}
input:focus {
  border-color: #007acc;
  outline: none;
}

Exemplo: Listrando tabela

tr:nth-child(even) {
  background-color: rgba(0, 0, 0, 0.05);
}

Exemplo: Posição e condições

.card:first-child {
  margin-left: 0;
}
.card:last-child {
  margin-right: 0;
}
input:not([type="submit"]) {
  background: #eef;
}
button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}

Pseudo-classes comuns

  • :hover, :active, :focus
  • :visited, :first-child, :last-child
  • :nth-child(n), :not(seletor), :disabled
